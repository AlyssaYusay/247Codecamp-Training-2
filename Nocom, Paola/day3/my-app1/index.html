CSS Selectors

  1. Element
  2. #Id - Unique identifier. Use # to call
  3. .Class - Group identifier. Use . to call
  4. * calls on everything
  5. commas - calls on multiple selectors
  6. Element + Class
  7. Element + ID

  2. :hover
  3. :active
  4. :focus
